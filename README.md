
# localfarm

## Description
`localfarm` is a web app designed to support local greenhouse agriculture using cutting-edge technologies like IoT, data visualization, and automation. It enhances farming efficiency by offering real-time monitoring and automated control of greenhouse conditions.

## Objectives
- **Real-Time Monitoring & Automation:** Use IoT sensors for continuous data and automated control systems.
- **Simplified Management:** Streamline greenhouse operations for greater efficiency and sustainability.

## Scope
- **Monitoring:** Continuous tracking of temperature, humidity, and soil moisture.
- **Control:** Automated systems for ventilation and irrigation.
- **Analytics & Visualization:** Data analysis and a user-friendly dashboard.
- **Alerts:** Automated notifications for critical conditions.
- **Implementation:** Hardware and software setup, testing, and maintenance.

## Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/yourusername/localfarm.git
   cd localfarm
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

## Scripts

- **Dev Server:** `npm run dev` - Start the Vite development server.
- **Build:** `npm run build` - Build the project for production.
- **Lint:** `npm run lint` - Run ESLint to ensure code quality.
- **Preview:** `npm run preview` - Preview the production build.
- **JSON Server:** `npm run server` - Start a JSON server to serve mock data.
